# Competitive Landscape

## Direct Competitors

| App | Category | Strength | Gap Momentum Fills |
|-----|----------|----------|-------------------|
| Griply | Life OS | Vision → Goals hierarchy | No AI, no native desktop, clunky UX |
| Sunsama ($20/mo) | Daily planner | Calendar-first, ritual-based | No AI automation, no free tier, weak mobile, no pillar layer |
| Strides | Habit tracker | Simple streaks, life area tags | No goal hierarchy, no AI, no task management, iOS only |
| Reclaim.ai | Calendar AI | Auto-schedules habits on calendar | No goal tracking, no journaling, Google Calendar only |
| TickTick | Task + habits | Task lists + habit tracking + calendar | No goal hierarchy, no AI coaching, no identity layer |
| Habitica | Gamified habits | RPG gamification, group challenges | No goal planning, no reviews, no AI, novelty wears off |
| GoalsOnTrack | Goal planner | SMART goals, sub-goals, journal | Dated UI, no AI, no native apps, no habit streaks |
| Todoist | Task management | Clean UX, natural language, integrations | No goal hierarchy, no habits, no AI coaching |
| Notion | Flexible workspace | Build anything with databases | Not opinionated, requires setup, no coaching, no streaks |

## Momentum's Unique Combination
No single competitor has all of these:
1. Life Pillars (identity-level framing)
2. AI conversational goal creation
3. Habit tracking with streaks
4. Micro-journaling with AI integration
5. Daily coaching
6. Native cross-platform
7. Free tier

Griply is closest structurally (#1, #3) but has no AI (#2, #4, #5) and no native desktop (#6).
